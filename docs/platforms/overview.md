# Platforms Overview

NanoMind currently spans three practical platform surfaces:

- browser control UI
- Rust edge host
- ESP32-S3 device client

## Pages

- [Browser Control UI](./browser-control-ui.md)
- [ESP32-S3 Device Client](./esp32-s3.md)

## Host Notes

- The browser UI runs well on desktop and can adapt down to mobile.
- The Rust edge server is intended for a local laptop, desktop, or small server.
- The ESP32 firmware assumes an ESP32-S3 class device and Wi-Fi connectivity.
